FAQ

  1. What language should I learn first?

    I will state that it is MY OPINION that a person should start learning programming with the language "Typescript". It is high level enough that you won't get bogged down by minute details, it uses a compiler and an interperator, it can be used server-side and client-side, it's widely supported by common engineering tools, etc.

  2. Is [insert any language] dead?

    No. It's still used. However you are probably asking this because you want to sit down and memorize a language. Instead you should be focusing on learning the topic that interests you, and you should learn/utilize the language that fits that as you develop. Programming does not revolve around a language. Languages are just tools to be used when needed.

  3. I'm learning but have no project ideas, what should I make?

    A blog web app that allows people to create their own accounts and manage blog posts via user interface (browser) and API. You will learn about front-end, back-end, database management, http requests, security authentication, server hosting, etc.
